# Global Weather View 🌤️

A beginner-friendly weather dashboard for learning open source contributions.

## What is this?

A real weather app that displays current conditions for 20 cities worldwide. Perfect for developers who want to start contributing to open source projects.

## Features

- 🌍 Weather for 20 global cities
- 🔍 Search cities by name/country
- 🌡️ Celsius/Fahrenheit toggle
- 📌 Pin favorite cities
- 📅 7-day forecasts

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:9002`


## Tech Stack

- Next.js 15 + TypeScript
- Tailwind CSS
- Open-Meteo API (free)
- Firebase Hosting
